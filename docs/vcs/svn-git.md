# ICR Conversion from SVN to GitLab

This page is specifically targetted at those in the ICR migrating from svn to GitLab and the workflow that can map from 1 to the other as seemlessly as possible.

Commands used:
- **svn diff**  
- **svn add**  
- **svn remove**  
- **svn delete**  
- **svn rename**  
- **svn move**  
- **svn checkout**  
- **svn commit**  
- **svn update**  
- **svn log**
